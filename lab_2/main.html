Welcome to the Sumac Dynamic Webpages Application

This application provides an interactive way to explore posts, user profiles, and comments dynamically.

Purpose

The goal of this application is to showcase dynamic content fetching and display using modern web technologies.

Features

  • View dynamically loaded posts with user information.
  • Read and interact with user comments.
  • Responsive navigation for easy access.
  • Interactive user profiles.
  • Contact form validation.
